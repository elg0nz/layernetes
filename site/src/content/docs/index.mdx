## From your laptop to a public URL, in one push.

Building the agent is the fast part now. Layernetes is everything after: the
container, the registry, the secrets, the ingress, the URL, the TLS. It already
runs. You write the agent and push.

```sh
llnate init my-agent      # scaffold your agent plus everything the cloud expects
cd my-agent
llnate plugin install     # wire up your AI coding assistant
# ... build your agent live, the way you build everything now ...
llnate login              # claims your cloud repo and encryption keys
llnate keys               # encrypt your API keys into the repo, safely
llnate push               # deploy: streams progress, prints your live URL
```

`push` blocks until your agent is live, then prints its public URL. Paste it into a
demo, drop it in a channel, or point Claude at it. No dashboard, no cloud account.
